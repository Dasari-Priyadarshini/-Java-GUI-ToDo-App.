## Project Description
This is a simple Java desktop application built using **Swing**.  
The app allows users to **add** and **delete** tasks from a to-do list through a graphical user interface.

## Objective
To learn how to build GUI-based applications in Java using Swing components and event handling.

## Tools & Technologies
- Java (JDK 8 or higher)
- Swing (built-in Java GUI library)
- IntelliJ CE / Eclipse / VS Code

## Features
- Add new tasks
- Delete selected tasks
- Simple and user-friendly interface

## How It Works
1. Type a task in the text field.
2. Click **Add Task** to insert it into the list.
3. Select any task from the list.
4. Click **Delete Task** to remove it.
